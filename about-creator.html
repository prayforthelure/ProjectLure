CREATOR / 制作者について

Lure / 擬似餌 🎣 として活動しています。
Project LURE は、色から生まれたキャラクターたちの世界を通して、日々の悩みや揺らぎにそっと寄り添える作品を作りたいという思いから始まりました。

このページでは、「どんな姿勢で創作しているのか」をまとめています。

PERSONAL THEME

私が使っている「LURE / 擬似餌🎣」という名前は、擬似餌（ルアー）に由来しています。
本物ではなくても、色や形を自由に変えながら、水の中で魅力として機能する存在。その「変幻自在でいられること」への憧れが、活動名の根本にあります。

英語の lure が「惹きつける」という意味を持つように、決まった姿に固まらず、表現そのものを通して人を魅了できたらという思いも込めています。

また、漢字表記の「瑠曖（るあ）」には、光を帯びていながら、輪郭が曖昧でつかみきれない存在でいたいという願いがあります。
明確な形に縛られず、曖昧さの中に自由があるという考え方です。

本物の魚のように泳ぐ必要はなく、
“本物ではないからこそできる表現” を活かして誰かを惹きつける。
その姿勢が ぎじえプロジェクト の基盤となっています。

PROFILE

・名前：Lure / 擬似餌 🎣
・主な活動分野：キャラクターデザイン / 作曲 / ロゴ・作字
・好きな言葉：コンセプト

色をモチーフにしたキャラクターデザインと、彼らの心情を伝えられるキャラソンを主軸に活動しています。

CREATIVE STANCE

ぎじえプロジェクト では、ひとつのキャラクターを表現するために様々なメディアを使用しています。
イラスト・テキスト(小説・詩)・音楽を通して、世界を多面的に見せたいと考えています。

私はどの分野も初心者で、現在も勉強しながら制作を積み重ねています。
ですがどのメディアでも、見た人の心を揺さぶれるようなものをつくりたいという信念があります。

PERSONAL THEME

私にとって ぎじえプロジェクト は、創作のための場所であると同時に、
自分自身の感情や不安、葛藤を安全に外へ出すための装置のような存在でもあります。

イラストや音楽というフィルターを通すことで、
直接言葉にしづらい気持ちを変換し、整理することができます。
誰かの悩みや孤独と響き合う“ささやかな救い”になれば嬉しいと思っています。
