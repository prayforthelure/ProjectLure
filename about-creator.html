CREATOR / 制作者について

Lure / 擬似餌 🎣 として活動しています。
このページでは、名前の由来や創作に対する考え方をまとめています。

PROFILE / プロフィール

名前：Lure / 擬似餌 🎣
活動内容：キャラクターデザイン / イラスト / 作曲 / ロゴ・作字

色を手がかりにキャラクターを設計し、ビジュアルと言葉と音で提示する制作を続けています。

ORIGIN / 名前の由来

「Lure / 擬似餌🎣」という名前は、釣りの擬似餌（ルアー）に由来しています。
様々な色や形、用途があり、ひとつの見た目に縛られないこと。その変幻自在さに憧れています。

また英語の lure には「魅了する」という意味があります。
私の創作物で人々を惹きつけられるように、という願いを込めています。

STANCE / 創作方針

私がこのプロジェクトで重視しているのは、「キャラクターの原案を作ること」です。
価値観・悩み・生き方まで含めて設計し、その人物がどんな世界を見ているのかを作品として提示したいと考えています。

ぎじえプロジェクトでは、イラスト・テキスト（あらすじ/詩）・音楽など、 複数のメディアを使ってキャラクターを提示しています。
ただし、すべてを理解してもらうことは前提にしていません。

ひとつの作品を点として楽しむことも、いくつかを並べて線として見ることもできる。
その自由度を残したまま、作品として成立する形を積み重ねていきます。
