PERSONAL THEME

私が使っている「LURE / ルアー」という名前は、擬似餌（ルアー）に由来しています。
本物ではなくても、色や形を自由に変えながら、水の中で魅力として機能する存在。
その「変幻自在でいられること」への憧れが、活動名の根本にあります。

英語の lure が「惹きつける」という意味を持つように、
決まった姿に固まらず、表現そのものを通して人を魅了できたらという思いも込めています。

また、漢字表記の「瑠曖（るあ）」には、
光を帯びていながら、輪郭が曖昧でつかみきれない存在でいたいという願いがあります。
明確な形に縛られず、曖昧さの中に自由があるという考え方です。

本物の魚のように泳ぐ必要はなく、
“本物ではないからこそできる表現” を活かして誰かを惹きつける。
その姿勢が Project LURE の基盤となっています。

PERSONAL THEME

私が使っている「LURE / ルアー」という名前は、擬似餌（ルアー）に由来しています。
本物ではなくても、色や形を自由に変えながら、水の中で魅力を発揮する存在。
その「変幻自在でいられること」への憧れが、活動名の根底にあります。

同時に、英語の lure（惹きつける）という意味も重ねています。
決まった姿に固定されず、表現そのものを通して人を魅了したいという願いを込めています。

漢字表記の「瑠曖（るあ）」には、
光を帯びていながら輪郭が曖昧で、どこまでも形を変えられる存在でいたいという意味があります。
明確さよりも、揺らぎや曖昧さの中に自由を見つけたいという考え方です。

本物の魚のように泳ぐ必要はなく、
“本物ではないからこそ届くもの” を表現に変えて誰かを惹きつけたい。
そうした姿勢が Project LURE 全体の根にあります。

PROFILE

・名前（活動名）：LURE / 瑠曖（るあ）
・主な活動分野：キャラクターデザイン / 作曲（キャラソン） / テキスト / ロゴ・作字
・好きなモチーフ：色、職業、言葉、矛盾や葛藤のあるキャラクター

色を起点にしたキャラクターの造形と、
その心情を音として翻訳する「キャラソン制作」を主軸に活動しています。

CREATIVE STANCE

Project LURE では、ひとつのキャラクターを一度に完成させるのではなく、
「ビジュアル・言葉・音」を少しずつ積み重ねて育てる という方法をとっています。

立ち絵、短いテキスト、曲の仮歌……。
断片を順に積み上げることで、そのキャラクターの価値観や生き方が立ち上がっていきます。
完成品を一気に提示するのではなく、“育つ過程そのもの”を大事にしています。

PERSONAL THEME

私にとって Project LURE は、創作のための場所であると同時に、
自分自身の感情や不安、葛藤を安全に外へ出すための装置のような存在でもあります。

色やキャラクターというフィルターを通すことで、
直接言葉にしづらい気持ちを物語や音に変換し、整理することができます。
その断片が、誰かの悩みや孤独と響き合う“ささやかな救い”になれば嬉しいと思っています。
