Terms / 用語説明

Project LURE の中で頻出する専門用語や略語を、
まとめて確認できる用語集です。

アーク（Arc）やシリーズ（大テーマ）の一覧は、
JSONデータ（arcList.json / series.json）から自動生成しています。

ARC / アーク

アーク（Arc）は、キャラクターの内面や役割を象徴する属性ラベルです。
戦闘の有利不利を決めるゲーム的な属性ではなく、 そのキャラがどんな方向の「物語」を持っているかを整理するための分類として使います。

※ アークは arcList.json の内容をもとに自動表示しています。 キーワードを変更したい場合は、JSON側を編集してください。

CODE / コード

各キャラクターには、3桁の英数字コードが割り振られます。
例：000 / 004 / 40A など。

コードは、
「どのシリーズに属するか」「その中で何番目のキャラクターか」などを
構造的に表すために使います。

そまりものがたり（0_Occupation）など、シリーズごとのコードルールは、
データが固まり次第、別途まとめていきます。
