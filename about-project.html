Project / ぎじえプロジェクトとは

「ぎじえプロジェクト / Project LURE」は、
「色」をモチーフにしたキャラクターと、そのキャラクターたちの断片的な物語・楽曲・ビジュアルを 組み合わせて楽しむ創作プロジェクトです。

ここでは、プロジェクト全体のコンセプトや、シリーズ構成のざっくりとした見取り図をまとめています。

CONCEPT

Project LURE のいちばん大きな軸は「色 × キャラクター」です。
ひとつの色に対して、コード（番号）・職業や属性・テーマソング候補・キャッチコピーなどを設定し、 ひとりのキャラクターとして立ち上げていきます。

小説や長編ストーリーを主軸にするのではなく、ビジュアルとキャラソンを中心に、 断片的なテキストや設定資料で世界を補うスタイルを想定しています。

SERIES

Project LURE の中には、いくつかの「シリーズ（大テーマ）」があります。
たとえば、職業や立場を扱う 0_Occupation / そまりものがたり のほか、 感情や欲望を扱う 1_Instinct / こころものがたり などです。

シリーズごとにコード体系やテーマが決まっており、
「どのキャラがどのシリーズに属しているか」で、そのキャラが背負っている物語の方向性が変わります。

シリーズ一覧や大テーマの詳しい説明は、
Terms / 用語説明 で整理する予定です。

GOAL & FUTURE

当面の目標は、各キャラクターの立ち絵と、対応するキャラソン（またはイメージ楽曲） を順番に制作し、 ギャラリーや配信プラットフォームで見られる形にしていくことです。

それぞれのキャラクターは「単発キャラ」で終わらせず、
企画・展示・グッズ・コラボレーションなど、さまざまな形で再利用できるIP（キャラクター資産）として 育てていくことを目標にしています。
